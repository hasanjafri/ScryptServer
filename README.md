# Scrypt Server

## Usage in dev environment

* clone repo

```
git clone https://github.com/hasanjafri/ScryptServer.git
```

* install node modules

```
cd ScryptServer

npm install
```

* start the server in dev

```
npm run start
```

## Usage in production

* clone repo

```
git clone https://github.com/hasanjafri/ScryptServer.git
```

* install node modules

```
cd ScryptServer

npm install
```

* build and setup production server

```
npm run production
```
